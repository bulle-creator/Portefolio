RPG Maker


Histoire


Le RPG Maker est un logiciel qui peut être utilisé pour développer facilement des jeux RPG (Role Playing Game), même pour les personnes qui ne sont pas expérimentées dans la conception de jeux (création de jeux). Jusqu'en 2003, le programme a été créé par la société japonaise ASCII. Dès lors, Enterbrain a poursuivi le développement. Le RPG Maker dispose d'une interface utilisateur graphique (GUI) qui rend le fonctionnement très simple. Le logiciel contient de nombreuses options pour personnaliser un jeu. Dans chaque version se trouve un Run Time Package (RTP). Le RTP contient de nombreux graphiques et sons différents, bien que les fichiers créés par vous-même puissent être importés dans le programme tant qu'ils sont dans le bon format (taille, type de fichier, ...).
